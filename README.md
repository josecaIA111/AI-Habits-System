# 🧠 AI Habit System

AI-powered habit coach that analyzes your daily behavior and provides actionable feedback to improve discipline and consistency.

---

## 🚀 What is this?

AI Habit System is a simple but powerful tool that transforms your daily journal into structured insights.

Instead of generic motivation, it focuses on:
- identifying patterns
- detecting mistakes
- enforcing accountability
- building real discipline

---

## ⚙️ How it works

1. You write your daily log  
2. The AI analyzes your behavior  
3. You receive structured feedback:
   - productivity score  
   - main mistake  
   - action for tomorrow  
   - identity feedback  
   - behavioral patterns  

---

## 🧪 Features

- 🔥 Productivity scoring system  
- ⚡ Streak tracking  
- 📊 Progress visualization  
- 🎯 Actionable daily feedback  
- 🧠 Behavioral pattern analysis  
- 🎛️ Multiple modes (Soft / Standard / Hardcore)  

---
